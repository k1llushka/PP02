# Программа распознавания документов клиентов  
ООО «Авторы Брокеридж»

## Описание

Данное программное средство предназначено для автоматизации обработки документов клиентов. Программа выполняет распознавание текстовой информации из изображений документов и выводит результат пользователю.

Разработка выполнена в рамках производственной практики ПП.02 по специальности 09.02.07 «Информационные системы и программирование».

Система работает локально и не передаёт данные во внешние сети, что обеспечивает безопасность конфиденциальной информации.

---

## Возможности программы

- Распознавание текста из документов
- Поддержка изображений документов
- Извлечение текстовой информации
- Отображение результата распознавания
- Локальная работа без подключения к интернету
- Прототип корпоративной системы обработки документов

---

## Используемые технологии

- Python  
- RussianDocsOCR  
- SQLite

## Установка

1. Скачать или клонировать репозиторий:

```bash
git clone https://github.com/USERNAME/PP02.git
cd PP02
```

2. Создать виртуальное окружение:

```bash
python -m venv venv
source venv/Scripts/activate
```

3. Установить зависимости:

```bash
pip install -r requirements.txt
```

Запуск программы

```bash
python main.py
```

После запуска откроется окно программы, позволяющее выбрать документ и выполнить распознавание текста.# PP02
